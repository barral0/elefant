# Keyboard Shortcuts

A complete reference for all Emerald keyboard shortcuts.

---

## File operations

| Shortcut | Action |
|---|---|
| `Ctrl + S` | Save note |
| `Ctrl + D` | Download note as `.md` |

---

## Text formatting (editor)

| Shortcut | Action |
|---|---|
| `Ctrl + B` | **Bold** — wraps selection in `**...**` |
| `Ctrl + I` | *Italic* — wraps selection in `*...*` |
| `Ctrl + `` ` | Code — inline `` `...` `` or block ` ```...``` ` |

> **Tip:** Select text first, then apply a shortcut to wrap it. Without a selection, a placeholder is inserted.

---

## Editor actions

| Shortcut | Action |
|---|---|
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + X` | Cut |
| `Ctrl + C` | Copy |
| `Ctrl + V` | Paste (also detects image in clipboard) |

---

## Images

| Method | Action |
|---|---|
| `Ctrl + V` | Paste image from clipboard |
| Drag & drop | Drop image file onto editor pane |
| Right-click → Insert Image | Opens file picker |
| ⋮ menu → Insert Image | Same |

---

## Navigation

| Shortcut | Action |
|---|---|
| `?` | Open keyboard shortcuts help |
| `Esc` | Close any open modal or menu |
| Right-click in editor | Open formatting context menu |
| Right-click on file/folder | Open file browser context menu |

---

## Right-click context menu (editor)

| Option | Action |
|---|---|
| Undo | Undo last change |
| Redo | Redo last undone change |
| Cut / Copy / Paste | Clipboard operations |
| Bold / Italic / Code | Format selected text |
| Link | Insert `[text](url)` |
| Insert Image | Open image file picker |

---

## Right-click context menu (file browser)

| Option | Action |
|---|---|
| Rename | Focus the title input for editing |
| Delete | Delete the note or folder |
